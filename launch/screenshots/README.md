# Screenshots

`benchmark-output.txt` — the captured output of `npm run benchmark`. On launch day, paste the FIDELITY / NETWORK FOOTPRINT blocks into [carbon.now.sh](https://carbon.now.sh) (theme: One Dark or similar dev-friendly), download as PNG, save here as `benchmark-screenshot.png`. Use as the lead-tweet image.

The PNG isn't committed — it's regenerated per launch from the live capture so the styling stays in your control.
